# HorseShare

Project for the UTCN CTI third year Software Engineering course.

This is an uber clone using horses and carriages instead of cars and scooters.



## How to Run the Project
Prerequisites: Ensure that you have Docker and Docker Compose installed.

1 Clone the repository.

2 Create a .env file in the root directory and add the necessary environment variables for Firebase API keys and other configurations.

3 Run the command docker-compose up --build from the root directory.

4 Access the frontend at http://localhost:5173. The backend will be running at http://localhost:8000.
